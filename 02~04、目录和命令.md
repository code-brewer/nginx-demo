## 02、默认目录

1、nginx安装文件目录

```
/usr/local/Cellar/nginx
```

2、nginx配置文件目录

```
/usr/local/etc/nginx
```

3、config文件目录

```
/usr/local/etc/nginx/nginx.conf
```

4、系统hosts位置

```
/private/etc/hosts
```

5、查看目录的方法：

示例：打开nginx安装目录：

命令行输入：``/usr/local/Cellar/nginx``，再输入 ``open .``，即可打开finder。

## 03、nginx常用命令

```
nginx  #启动nginx
nginx -s quit  #快速停止nginx
nginx -V #查看版本，以及配置文件地址
nginx -v #查看版本
nginx -s reload|reopen|stop|quit   #重新加载配置|重启|快速停止|安全关闭nginx
nginx -h #帮助
```
