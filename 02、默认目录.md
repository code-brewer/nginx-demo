## 02、默认目录

1、nginx安装文件目录

```
/usr/local/Cellar/nginx
```

2、nginx配置文件目录

```
/usr/local/etc/nginx
```

3、config文件目录

```
/usr/local/etc/nginx/nginx.conf
```

4、系统hosts位置

```
/private/etc/hosts
```

5、查看目录的方法：

示例：打开nginx安装目录：

命令行输入：``/usr/local/Cellar/nginx``，再输入 ``open .``，即可打开finder。
